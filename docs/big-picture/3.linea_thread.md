# I thread e la linea di esecuzione

!!! Question "Domanda"
    Cosa succede alla linea di esecuzione quando ho più di un solo thread?

!!! warning "Importante"
    Ogni processo ha almeno un thread, detto main-thread.

---

Per capire cosa succede alla linea di esecuzione vediamo il seguente esempio:

```python
import threading, from time import sleep

def thread_1():
    while True:
        print("Dentro il thread")
        sleep(1)

t1 = threading.Thread(target=thread_1)
t1.start()

while True:
    print("Dentro il main-thread....")
    sleep(1)
```

In pratica questo script crea un thread oltre il sempre-presente main-thread.

Vediamo la sequenza della linea di esecuzione, si parte dalla linea #1:


Qui viene creato un nuovo thread oltre al main-thread.

!!! warning "Importante"

    Il primo sforzo per capire i thread avviene in questo momento.

Da ora in poi sappiamo con **^^certezza^^** che:

- Il main-thread eseguirà ripetutamente la sequenza (#11, #12, #13).
- Il thread-1 eseguirà ripetutamente le sequenza (#4, #5, #6).


=== "main-thread"


=== "thread-1"

    ```python linenums="1" hl_lines="4-6"

    t1 = threading.Thread(target=thread_1)
    ```

!!! danger "Moltiplicazione della linea di esecuzione"
