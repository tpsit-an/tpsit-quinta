# Linee di esecuzione multiple

In questo caso abbiamo più linee di esecuzione contemporaneamente, una per ciascun thread:


=== "main-thread"

    ```python linenums="1" hl_lines="11"
    import threading, from time import sleep

    def thread_1():
        while True:
            print("Dentro il thread")
            sleep(1)

    t1 = threading.Thread(target=thread_1)
    t1.start()

    while True:
        print("Dentro il main-thread....")
        sleep(1)
    ```

=== "thread-1"

    ```python linenums="1" hl_lines="4"
    import threading, from time import sleep

    def thread_1():
        while True:
            print("Dentro il thread")
            sleep(1)

    t1 = threading.Thread(target=thread_1)
    t1.start()

    while True:
        print("Dentro il main-thread....")
        sleep(1)
    ```

!!! danger "Prevedibilità della prossima linea di esecuzione"
    Quando si hanno più thread, e quindi più linee di esecuzione, {==^^diventa IMPOSSIBILE prevedere la PROSSIMA linea di esecuzione^^==}.

La scelta dipende dalla libreria di threading, la quale:

- Permetterà a ciascun thread di eseguirsi.
- Ma non permette di prevedere quale sarà la prossima linea di esecuzione ad essere eseguita.

Partiamo dall'esempio precedente:

- Il main-thread deve eseguire la linea #11.
- Il thread-1 deve eseguire la linea #4.

Quindi abbiamo due linee di esecuzione:

```python linenums="1" hl_lines="4 11"
import threading, from time import sleep

def thread_1():
    while True:
        print("Dentro il thread")
        sleep(1)

t1 = threading.Thread(target=thread_1)
t1.start()

while True:
    print("Dentro il main-thread....")
    sleep(1)
```

In questa situazione:

- **^^Non possiamo prevedere quale delle due linee sarà eseguita^^**.

E poniamo che sia stata eseguita la linea #4 del main-thread, che ora è alla linea #5:

```python linenums="1" hl_lines="4 12"
import threading, from time import sleep

def thread_1():
    while True:
        print("Dentro il thread")
        sleep(1)

t1 = threading.Thread(target=thread_1)
t1.start()

while True:
    print("Dentro il main-thread....")
    sleep(1)
```

Anche in questa nuova situazione, non possiamo prevedere quale delle due linee verrà eseguita.
