Il protocollo HyperText Transfer Protocol (`HTTP`) è un protocollo applicazione utilizzato da due operatori (*client* e *server*) per eseguire operazioni su una risorsa definita dalla `URL`.

2. [Protocollo HTTP(s)](http.md#2-protocollo-https) — request, response, body, status code
3. [Metodi](http.md#3-metodi) — GET, POST
