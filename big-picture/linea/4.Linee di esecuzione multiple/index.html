    import threading, from time import sleep

def thread_1():
    while True:
        print("Dentro il thread")
        sleep(1)

t1 = threading.Thread(target=thread_1)
t1.start()

while True:
    print("Dentro il main-thread....")

    Prevedibilità della prossima linea di esecuzione

    Quando si hanno più thread, e quindi più linee di esecuzione, diventa IMPOSSIBILE prevedere la PROSSIMA linea di esecuzione.

    La scelta dipende dalla libreria di threading, la quale:

    • Permetterà l'esecuzione a ciascun thread.

    • Ma non permetterà di prevedere quale sarà la prossima linea di esecuzione.

    • Il main-thread deve eseguire la linea #11.

    • Il thread-1 deve eseguire la linea #4.

    Sequenza di esecuzione: linea 1

    Facciamo il caso che thread-1 acquiserà l'utilizzo della CPU eseguendo la linea #5. Allora:

    • La linea di esecuzione di thread-1 avanzerà.
    • La linea di esecuzione di main-thread rimane ferma.
