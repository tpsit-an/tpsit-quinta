    I thread e la linea di esecuzione

    Domanda

    Cosa succede alla linea di esecuzione quando ho più di un solo thread?

    Importante

    Ogni processo ha almeno un thread, detto main-thread.

    Per capire cosa succede alla linea di esecuzione vediamo il seguente esempio:

    import threading, from time import sleep

def thread_1():
    while True:
        print("Dentro il thread")
        sleep(1)

t1 = threading.Thread(target=thread_1)
t1.start()

while True:
    print("Dentro il main-thread....")
    sleep(1)

    In pratica questo script crea un thread oltre il sempre-presente main-thread.
